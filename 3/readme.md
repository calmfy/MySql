# 📊 Задание 3 — SQL-запросы к таблице Pet

## 📌 Описание

В рамках задания были написаны и отлажены SQL-запросы к таблице `pet` базы данных `pet_db`.

Цель:

* освоить базовые операции SQL
* научиться извлекать, фильтровать и агрегировать данные

---

## 🗄 Используемая таблица

Основная таблица:

* `pet` — информация о питомцах

Дополнительно:

* `owner` — владельцы животных

---

## 🚀 Выполнение запросов

Перед выполнением:

```sql
USE pet_db;
```

---

## 📋 Запросы

### 1. Данные на Партизана

```sql
SELECT *
FROM pet
WHERE name = 'Партизан';
```

📌 Получение полной информации о питомце по кличке.

---

### 2. Клички и породы с сортировкой по возрасту

```sql
SELECT name, breed, age
FROM pet
ORDER BY age;
```

📌 Вывод списка питомцев с сортировкой по возрасту.

---

### 3. Питомцы с описанием

```sql
SELECT *
FROM pet
WHERE description IS NOT NULL;
```

📌 Выборка питомцев, у которых заполнено описание.

---

### 4. Средний возраст пуделей

```sql
SELECT AVG(age) AS avg_age
FROM pet
WHERE breed = 'Пудель';
```

📌 Вычисление среднего возраста питомцев породы пудель.

---

### 5. Количество владельцев

```sql
SELECT COUNT(*) AS owner_count
FROM owner;
```

📌 Подсчет общего числа владельцев.

---

### 6. Количество питомцев каждой породы

```sql
SELECT breed, COUNT(*) AS count_pets
FROM pet
GROUP BY breed;
```

📌 Группировка питомцев по породе.

---

### 7. Породы с количеством питомцев больше одного

```sql
SELECT breed, COUNT(*) AS count_pets
FROM pet
GROUP BY breed
HAVING COUNT(*) > 1;
```

📌 Отбор пород, представленных более чем одним питомцем.

---

## ✍️ Дополнительные запросы

### 8. Использование BETWEEN

```sql
SELECT *
FROM pet
WHERE age BETWEEN 2 AND 5;
```

📌 Питомцы в возрасте от 2 до 5 лет.

---

### 9. Использование LIKE

```sql
SELECT *
FROM pet
WHERE name LIKE 'Б%';
```

📌 Питомцы, чьи клички начинаются на букву "Б".

---

### 10. Использование IN

```sql
SELECT *
FROM pet
WHERE breed IN ('Пудель', 'Овчарка', 'Сиамская');
```

📌 Питомцы выбранных пород.
