# 📊 Задание 3 — SQL-запросы к таблице Pet

## 📌 Описание

В рамках задания были написаны и отлажены SQL-запросы к таблице `pet` базы данных `pet_db`.

Цель задания:

* освоить базовые SQL-запросы
* научиться фильтровать, сортировать и агрегировать данные

---

## 🗄 Используемые таблицы

* `pet` — информация о питомцах
* `owner` — информация о владельцах

---

## 🚀 Подключение к базе

```sql
USE pet_db;
```

---

## 📋 Основные запросы

### 1. Данные на Partizan

```sql
SELECT *
FROM pet
WHERE Nick = 'Partizan';
```

📌 Получение полной информации о питомце по кличке.

---

### 2. Клички, породы и возраст (сортировка по возрасту)

```sql
SELECT Nick, Breed, Age
FROM pet
ORDER BY Age;
```

📌 Список всех питомцев, отсортированный по возрасту.

---

### 3. Питомцы с описанием

```sql
SELECT *
FROM pet
WHERE Description IS NOT NULL;
```

📌 Выборка питомцев, у которых заполнено описание.

---

### 4. Средний возраст пуделей

```sql
SELECT AVG(Age) AS avg_age
FROM pet
WHERE Breed = 'Poodle';
```

📌 Расчёт среднего возраста питомцев породы пудель.

---

### 5. Количество владельцев

```sql
SELECT COUNT(*) AS owner_count
FROM owner;
```

📌 Подсчёт общего числа владельцев.

---

### 6. Количество питомцев каждой породы

```sql
SELECT Breed, COUNT(*) AS count_pets
FROM pet
GROUP BY Breed;
```

📌 Группировка питомцев по породам.

---

### 7. Породы, представленные более чем одним питомцем

```sql
SELECT Breed, COUNT(*) AS count_pets
FROM pet
GROUP BY Breed
HAVING COUNT(*) > 1;
```

📌 Отбор пород с количеством питомцев больше одного.

---

## ✍️ Дополнительные запросы

### 8. Использование BETWEEN

```sql
SELECT *
FROM pet
WHERE Age BETWEEN 2 AND 5;
```

📌 Питомцы в возрасте от 2 до 5 лет.

---

### 9. Использование LIKE

```sql
SELECT *
FROM pet
WHERE Nick LIKE 'B%';
```

📌 Питомцы, чьи клички начинаются на букву "B".

---

### 10. Использование IN

```sql
SELECT *
FROM pet
WHERE Breed IN ('Poodle', 'Shepherd', 'Siamese');
```

📌 Питомцы заданных пород.
